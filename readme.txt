Các bước cài đặt : 

- Cài đặt xampp, để tạo server local tải tại : https://www.apachefriends.org/download.html
- chạy Apache của xampp
- giải nén và copy thư mục application.rar đến folder :\xampp\htdocs
- copy toàn bộ file cần tìm kiếm vào thư mục "folder_file_input"
- đặt tên file có ký tự tìm kiếm là file_name_search.txt
- chạy trên local : http://localhost/application/
- click button tìm kiếm.

=> chương trình trả kết quả về tại folder: folder_file_output
